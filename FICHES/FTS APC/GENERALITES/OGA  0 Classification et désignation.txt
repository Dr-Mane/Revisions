#FFR2022.01

Les armes sont classifiées en fonction de :

- Leur destination au combat
- L'énergie de leur système moteur
- La manière dont s'effectuent le chargement et le mode de tir


Les différentes destinations au combat :

- Les armes de poing
- Les armes d'épaule
- Les mitrailleuses légères
- Les mitrailleuses lourdes
- Les armes antichar
- Les mortiers
- Les lances-grenades
- Les armes blanches


Les catégories d'armes d'épaule :

- Les pistolets-mitrailleurs
- Les carabines ou les mousquetons
- Les fusils
- Les fusils d'assaut
//
(+ Les Armes de Défense Rapprochée (ADR))


Les catégories de lance-grenades :

- Individuel
- Monté
- Modulaire
- Automatique


Les catégories d'armes blanches :

- Perforante 
- Contondante
- Tranchante


Les différents types d'énergie du système moteur :

- Armes non-automatiques
- Armes semi-automatiques
- Armes automatiques


Définition des armes non-automatiques :

Arme dont toutes les opérations du cycle de fonctionnement sont effectuées par la force musculaire du servant.


Définition des armes semi-automatiques et automatiques :

Arme dont toutes les opérations du cycle de fonctionnement sont assurées par l'énergie développée par les gaz.
//
Cependant, dans les armes semi-automatiques, un dispositif quelconque enraye le système de mise de feu pour ne permettre que le coup par coup. 


Les manières dont s'effectuent le chargement et le mode de tir :

- Armes non automatiques :
	- A chargement simple par la bouche
	- A chargement simple par l'arrière ou la chambre
	- A répétition
//
- Armes semi-automatiques
//
- Armes automatiques :
	- A tir continu
	- A tir mixte


Les éléments de la désignation d'une arme :

- Leur genre
- Leur calibre
- Leur modèle
- Éventuellement, une ou plusieurs précisions supplémentaires
