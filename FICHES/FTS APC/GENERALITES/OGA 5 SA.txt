#FFR2022.01

Définition de l'alimentation :

Alimenter une arme, c'est lui fournir dans des conditions susceptibles d'assurer son fonctionnement normal, les munitions qu'elle doit tirer, et, assurer l'évacuation des résidus de tir (étuis).
L'alimentation comporte le chargement de l'arme et l'évacuation de l'étui.


Les opérations à effectuer pour assurer le chargement :

- L'emmagasinage ou garnissage
- L'approvisionnement
- Le transport
- La distribution
- La présentation
- L'introduction


Les types d'introduction :

- Introduction directe
- Introduction indirecte


// MAGASIN


Les types de magasins :

- Intégré à l'arme
- Indépendant de l'arme


Les magasins intégrés à l'arme :

- Dans la boîte de culasse : 
	- A chargeur introduit
	- A chargeur non introduit
- Tubulaire dans le fût
- Tubulaire dans la crosse


Les magasins indépendants de l'arme :

- Fixes pendant le tir
- Mobiles pendant le tir
- Particuliers


Les magasins fixes pendant le tir :

- Pile unique, appui sur deux lèvres
- Pile imbriquée, appui sur une lèvre
- Pile imbriquée, appui sur deux lèvres
- A pivotement de cartouche
- A tambour fixe


Les magasins mobiles pendant le tir :

- A tambour tournant 
- A bandes souples mues par le moteur général de l'arme :
	- A maillons métalliques détachables : ouverts ou fermés
 	- A maillons métalliques indétachables : ouverts ou fermés


Les différents types d'extracteurs :

- A action normale
- A action forcée
- A rainures de culasse


Les extracteurs à action normale :

- Lame ressort à griffe
- Extracteur axé et poussé par un ressort
- Extracteur à translation
- Extracteur à talon


Les extracteurs à action forcée :

- Extracteur Berdan
- Extracteur ENT


Les extracteurs à rainures de culasse :

- Simple
- Double


// ÉJECTION


Les différents types d'éjecteurs :

- Projetant l'étui hors de l'arme
- Abandonnant l'étui hors de l'arme


Les éjecteurs projetant :

- Fixe
- Effaçable
- Ayant un mouvement propre


Les éjecteurs fixes :

- Porté par la boîte de culasse ou le support de mécanisme
- Porté par le couloir d'alimentation


Les éjecteurs effaçables :

- Porté par la boîte de culasse
- Porté par la culasse


Éjecteur ayant un mouvement propre :

Éjecteur à ressort porté par la culasse
