#FFR2022.01


Les armes sont classifiées en fonction de :

- Leur destination au combat
- L'énergie de leur système moteur
- La manière dont s'effectuent le chargement et le mode de tir


Quelles sont les différentes destinations au combat ?

- Les armes de poing
- Les armes d'épaule
- Les mitrailleuses légères
- Les mitrailleuses lourdes
- Les armes antichars
- Les mortiers
- Les lance-grenades
- Les armes blanches


Quels sont les genres d'armes d'épaule ?

- Les pistolets mitrailleurs
- Les carabines (ou les mousquetons)
- Les fusils
- Les fusils d'assaut
//
(+ Les Armes de Défense Rapprochée (ADR))


Quels sont les genres de lance-grenades ?

- Individuel
- Monté
- Modulaire
- Automatique


Quels sont les genres d'armes blanches ?

- Perforante 
- Contondante
- Tranchante


Quels sont les différents types d'énergie du système moteur ?

- Arme non-automatique
- Arme semi-automatique et automatique


Quelles sont les manières dont s'effectuent le chargement et le mode de tir ?

- Armes non automatiques :
	- A chargement simple par la bouche
	- A chargement simple par l'arrière ou la chambre
	- A répétition
//
- Armes semi-automatiques
//
- Armes automatiques :
	- A tir continu
	- A tir mixte


Par quoi sont désignées les armes ?

- Leur genre
- Leur calibre
- Leur modèle
- Une ou plusieurs précisions supplémentaires


Quelles sont les deux machines qui forment les armes ?

- Une machine principale (le canon)
- Une machine auxiliaire (la culasse, associée à différents systèmes, dispositifs et mécanismes)
